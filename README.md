## Prerequisites

1. Node.js (v18.16.0)
2. Python (v3.11)
3. Pip (v22.3.1)

## Installing Dependencies

For Windows users, run the following commands in Powershell:

```
npm install; if ($?) { pip install -r requirements.txt }
```

In command prompt:

```
npm install && pip install -r requirements.txt
```
